# Lootbox Randomizer

A simple pseudo-random reward system to simulate loot boxes in Web3 games.

## Features
- On-chain reward emission
- Pseudo-random number generation
- Lightweight Solidity logic

## Technologies
Solidity  
ethers.js  
Base testnet
